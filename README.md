# SmartSpend

A modern, intuitive expense tracking application built with React and Vite. SmartSpend helps you manage your finances effortlessly with real-time analytics, smart categorization, and data persistence.

## Features

✨ **Expense Management**
- Add, view, and delete expenses easily
- Real-time expense tracking with timestamps
- Automatic data persistence using browser localStorage

📊 **Analytics & Dashboard**
- Visual charts and graphs using Recharts
- Expense summaries and statistics
- Category-wise expense breakdown

🏷️ **Smart Categorization**
- 7 predefined categories: Food, Travel, Bills, Shopping, Entertainment, Health, Others
- Color-coded categories for quick identification
- Filter expenses by category

🔍 **Advanced Filtering & Search**
- Filter by category, date range, and amount range
- Search expenses by keywords
- Multiple sorting options (recent, oldest, highest, lowest)

💾 **Data Persistence**
- All expenses saved locally in browser storage
- No server required - complete privacy
- Works offline

## Tech Stack

- **React** 19.0.0 - UI library
- **Vite** 6.2.0 - Build tool and dev server
- **Tailwind CSS** 4.1.14 - Utility-first CSS framework
- **Recharts** 3.8.1 - Data visualization library
- **Lucide React** 0.546.0 - Icon library
- **Motion** 12.23.24 - Animation library
